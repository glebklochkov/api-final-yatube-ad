# API для социальной сети Yatube
## API интерфейс для социальной сети Yatube. Позволяет публиковать посты, комментировать их и подписываться на авторов.

### Как запустить проект:
Клонировать репозиторий и перейти в него в командной строке:

```
git clone https://github.com/glebklochkov/api-final-yatube-ad.git
```
```
cd api_yatube
```

Cоздать и активировать виртуальное окружение:
```
python3 -m venv env
```
```
source env/bin/activate
```

Установить зависимости из файла requirements.txt:

```
python3 -m pip install --upgrade pip
```
```
pip install -r requirements.txt
```

Выполнить миграции:
```
python3 manage.py migrate
```

Запустить проект:
```
python3 manage.py runserver
```

Примеры запросов к API:
Получение токена:
```
POST /api/v1/api-token-auth/
```

Передайте логин и пароль в формате JSON:
```
{
    "username": "имя_пользователя",
    "password": "ваш_пароль"
}
```

Все последующие запросы должны содержать заголовок:

Authorization: Token <ваш_токен>
Получение списка постов:
```

GET /api/v1/posts/
```

Создание нового поста:

```
POST /api/v1/posts/
```

Пример тела запроса для создания поста:
```
{
    "text": "Текст вашего поста",
    "group": 1
}
```

Добавление комментария к посту:
```
POST /api/v1/posts/{post_id}/comments/
```

Пример тела запроса для комментария:
```

{
    "text": "Ваш комментарий"
}
```

Подписка на автора:
```
POST /api/v1/follow/
```

Пример тела запроса для подписки:
```
{
    "following": "имя_автора"
}
```
